I want to formally represent latents of (probabilistic) computable functions

so I came up with something like this: 
- the space we're working in is a presented Markov category $(\mathcal{C}, \times, I, T)$ in which morphisms are generated by $T$, which we call the set of atomic morphisms
- a derivation $D \in D_f$ of $f: A \to B$ is a string diagram built inductively from $T$ using composition, tensoring, and copy/discard
	- concretely, let $S$ be the set of morphisms generated by $T$ plus copy and discard morphisms and closed under tensoring
	- a derivation is a finite sequence of morphisms from $S$ that compose to $f$
- the set of latents $L_D$ of $D$ are all latents $A \to -$ constructible from morphisms in the derivation and the same 4 operations. for laziness sake we essentially treat latents as general random variables not just hidden variables
- two latents $f: A \to B$ and $g: A \to C$ are equivalent iff they are equivalent in the coslice category $A/\mathcal{C}$
- we define an important concept which is essential latents: a set of latents $L \in \phi(f)$ is essential iff for every derivation $D \in D_f$, $L_D \cap L \neq \varnothing$
- minimal essential latent set $L \in \phi(f)$ cannot be reduced further without being non-essential
